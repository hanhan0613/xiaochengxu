# 学生签到小程序

基于微信云开发的学生签到系统，支持 Excel 批量导入学生名单、二维码签到、实时查看签到状态。

## 功能概述

- **管理后台**：Excel 一键上传学生名单（姓名+手机号），支持手动添加、重置签到、清空数据
- **学生端**：输入姓名和手机号验证身份，生成签到二维码
- **教师端**：扫描学生二维码完成签到，实时查看全部学生签到状态和统计数据

## 部署步骤

### 1. 准备工作

1. 注册微信小程序账号，获取 AppID
2. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

### 2. 导入项目

1. 打开微信开发者工具
2. 选择「导入项目」
3. 选择本项目文件夹 `ticket-miniapp`
4. 填入你的 AppID
5. 点击「导入」

### 3. 开通云开发

1. 在开发者工具中点击「云开发」按钮
2. 开通云开发环境，记下环境 ID
3. 打开 `app.js`，将 `your-env-id` 替换为你的云开发环境 ID

### 4. 创建数据库集合

在云开发控制台 -> 数据库中，创建集合：

- **students** — 学生数据表

设置数据权限为「所有用户可读，仅创建者可读写」或根据需要调整。

### 5. 上传云函数

在开发者工具中，右键点击 `cloudfunctions` 下的每个云函数文件夹，选择「上传并部署：云端安装依赖」：

- uploadStudents
- login
- checkIn
- getStudents

### 6. 关于 Excel 解析

项目内置了精简版 XLSX 解析器。如需更完整的 Excel 支持，建议：

1. 在项目根目录运行：`npm install xlsx`
2. 在微信开发者工具中点击「工具」->「构建 npm」
3. 将 `admin.js` 中的引用改为：`const XLSX = require('xlsx')`

## Excel 文件格式要求

上传的 Excel 文件（.xlsx）需包含以下列：

| 姓名 | 手机号 |
|------|--------|
| 张三 | 13800138001 |
| 李四 | 13800138002 |

列名支持模糊匹配：「姓名/名字/name」和「手机/电话/号码/phone/tel」均可识别。

## 使用流程

1. **管理员**：进入管理后台，上传包含学生姓名和手机号的 Excel 文件
2. **学生**：进入学生端，输入已登记的姓名和手机号，生成二维码
3. **教师**：进入教师端，点击「扫码签到」扫描学生二维码，完成签到
4. **教师**：在教师端查看实时签到统计和每位学生的签到状态

## 注意事项

- 二维码有效期为 5 分钟，过期后学生需重新进入页面刷新
- 已签到的学生重复扫码会提示「已签到」
- 重置签到状态会将所有学生恢复为「未签到」
- 清空数据操作不可恢复，请谨慎使用
